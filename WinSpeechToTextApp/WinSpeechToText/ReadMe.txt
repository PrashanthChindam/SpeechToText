
*** Info ***
 * This application uses Microsoft Speech Services for converting speech to text.
 * Inorder to use this application, provide the subscription key and end point location in the config file. 
 * Refer config file for more configuartion details like 
   -- min & max recording time (in seconds)
   -- recorded files storage location, etc
==============================================================================================================================

*** Steps to get Microsoft Speech Service subscription key ***

 * Sign up at "https://azure.microsoft.com/en-us/try/cognitive-services/" to get Speech Serives subscription key.

 * Subscribed services can be found at "https://azure.microsoft.com/en-us/try/cognitive-services/my-apis/?apiSlug=speech-services"

 * Microsoft provides a free Speech Services trial subscription for 30 days.

 * Location: location can be found in the endpoint of speech services.
   For example, "westus" is the location of the below endpoint
   Endpoint: https://westus.api.cognitive.microsoft.com/sts/v1.0
